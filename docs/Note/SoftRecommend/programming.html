4. SQL 数据库

MySQL [Free]

官网 目前应用最广泛的 Free 数据库，长期老二。

Oracle

官网 排名第一的数据库，但是真心贵。

Microsoft SQL Server

官网 微软系必选，常年第三。

PostgreSQL [Free]

官网 据说是最先进，然而就那么回事，常年第四。

MariaDB [Free]

官网 完全兼容 MySQL，且为开源版本。

SQLite [Free]

官网 应用最广泛的文件型关系数据库。移动端上的不二选择。

还有一些用的非常少的数据库例如：IBM Db2，Microsoft Access 之类的就不写进来了。

5. NoSQL 数据库

MongoDB[社区版 Free]

官网 分布式文件存储的数据库。了解不多早期出现过数据丢失的严重 Bug，所以一直未深入了解。

Redis [Free]

官网 高性能 Key-Value 数据库。速度快，可以用于当缓存使用。

Key-Value 还有Memcached但是基本被Redis干翻了。
