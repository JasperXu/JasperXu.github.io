其他软件

1. 虚拟机

Windows macOS Linux

VMWare

Windows 下最好用的虚拟机。

Hyper-V [Free]

微软出品，系统自带的虚拟机。由于独占一直不兼容与其他虚拟机。

目前最新版本的系统中已经不是独占了，需要达到以下要求即可都开启。

Windows 10 20H1 内部版本 19041.264 或更新版本
VMware Workstation 15.5.5

Parallels Desktop

macOS 下最好用的虚拟机。

VMWare Fusion [Free]

免费版的需要去英文的官网申请，一次只能开启一台虚拟机，反正够用了。
